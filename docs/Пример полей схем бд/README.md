# Устаревший пример (не источник правды)

Файл `api.ts` — старый вывод **openapi-typescript** с другого бэка.

**Актуальные документы в репозитории:**
- **`docs/api/FRONTEND_CONTRACT.md`** — что **уже внедрено** во фронте (CRUD, поля под `domain` и `core/api`).
- **`docs/api/API_FUTURE_CHECKLIST.md`** — черновик эндпоинтов (импорт, загрузки, товары, КП…); перед кодом — согласовать и перенести в контракт.
- **`libs/domain/src/lib/**`** — модели.

**Можно удалить эту папку**, если backlog в чеклисте вас устраивает и архив `api.ts` не нужен.

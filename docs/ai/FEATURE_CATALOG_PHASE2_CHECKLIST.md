# Фаза 2 каталога — товары (чеклист)

**Цель:** `Product` на `/catalog/products`: обязательная категория, опционально тип детали и материал на карточке, быстрое создание справочников из формы (`*ForPicker` + `CatalogLookupService`).

## В репозитории

- **Domain:** `libs/domain/src/lib/products/`, модель `ProductSpecification` в `product-specifications/` (для будущего КП/матрицы).
- **API:** `products-api.service.ts`; `product-specifications-api.service.ts` — **каркас без UI** (нигде не инжектится).
- **Фича:** `features/products` — страница **четыре колонки** (категории | товары | тех. описание | материалы), клик по строке товара сужает боковые списки; повторный клик по той же строке или клик вне неё — полные списки (без отдельной панели «фильтр»).
- **Доки:** `FRONTEND_CONTRACT.md`, `API_FUTURE_CHECKLIST.md`.

## Отключено / отложено

- Страница **`/catalog/products/:id/specifications`**, `ProductSpecificationsService` и формы строк спецификации — **удалены**; при необходимости восстановить по `ProductSpecificationsApiService` + domain.

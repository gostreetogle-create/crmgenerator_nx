# Архив чек-листов фич

Тут лежат заархивированные чек-листы фич, которые:
- достигли критерия MVP (по `FEATURE_CHECKLIST_BASE.md`)
- больше не находятся в активной разработке

Назначение архива:
- чтобы не создавать дубликаты чек-листов одной и той же фичи
- чтобы быстро смотреть “что именно было сделано” в прошлом

Правило:
- если фича вернулась в работу как *новая итерация/версия*, создавай новый файл чек-листа в активной зоне (`docs/ai/`) и в `History` указывай связь с архивной версией.

# Архив чек-листов фич

Тут при необходимости лежат **заархивированные** чек-листы фич, которые:

- достигли критерия MVP (по `FEATURE_CHECKLIST_BASE.md`);
- больше не находятся в активной разработке.

Назначение архива:

- не плодить дубликаты чек-листов одной и той же фичи;
- хранить снимок «что было сделано» в прошлой итерации.

Правило:

- если фича вернулась в работу как *новая итерация*, создавай новый файл в **`docs/ai/FEATURE_<NAME>_CHECKLIST.md`** и в `History` укажи связь с архивной версией;
- **активный** чек-лист всегда в `docs/ai/`, не в архиве.

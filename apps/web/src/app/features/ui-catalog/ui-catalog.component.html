Размерный минимум: типографика 12/14/16, spacing-шкала 4/6/8/12, радиусы 4/6, кнопки 28/34/40, input 30/36/42, диалоги sm/md/lg/xl.

Почему старую витрину можно скрыть: она была демонстрационной и не является источником истины; истина — код компонентов в libs/ui-kit, токены в libs/design-tokens и артикулы в документации Eve.
